## What's Included

The Gigadrive SDK is a monorepo containing several focused packages to support your development workflow:

### Core Packages

<CardGroup cols={2}>
  <Card title="@gigadrive/sdk" icon="box">

  </Card>
  <Card title="@gigadrive/commons" icon="wrench">

  <Card title="@gigadrive/network-config" icon="gear">

## Key Features

<Steps>
  <Step title="TypeScript-First">
    Built with TypeScript for excellent type safety and IDE support. All packages include comprehensive type definitions.
  </Step>
  
  <Step title="Effect-TS Architecture">
    The CLI uses Effect-TS for end-to-end type safety, structured error handling, and composable dependency injection.
  </Step>
  
  <Step title="Modern Build System">
    Dual CJS/ESM output with conditional exports ensures compatibility across Node.js and modern bundlers.
  </Step>
  
  <Step title="Framework Detection">
    Automatic framework detection for Next.js, React, Vue, and more - simplifying deployment configuration.
  </Step>
</Steps>

## Philosophy

The Gigadrive SDK is designed with these principles in mind:

- **Developer Experience First** - Clear APIs, excellent TypeScript support, and comprehensive documentation
- **Composability** - Small, focused packages that work together seamlessly
- **Type Safety** - Leverage TypeScript's type system to catch errors at compile time
- **Modern Standards** - Built using current best practices and modern JavaScript features

## Getting Started

Ready to build with Gigadrive? Start with our [Quick Start guide](/quickstart) or jump straight to [Installation](/installation) to add the SDK to your project.

<Note>
  The Gigadrive SDK requires Node.js v18 or higher. Some packages (like the CLI) require Node.js v20+.
</Note>
