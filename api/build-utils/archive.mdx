## Overview

The `@gigadrive/build-utils` package includes `archiver` and `extract-zip` as dependencies, indicating planned support for archive operations.

<Note>
  Archive utilities are currently in development. This page documents the planned functionality based on the package dependencies.
</Note>

## Planned Features

Based on the package dependencies, the following archive operations are planned:

### Archive Creation

Using the `archiver` library (v7.0.1+), the package will support:

- Creating ZIP archives
- Creating TAR archives (with optional compression)
- Streaming archive creation for large file sets
- Adding files and directories to archives
- Custom compression levels

### Archive Extraction

Using the `extract-zip` library (v2.0.1+), the package will support:

- Extracting ZIP archives
- Streaming extraction for large archives
- Custom output directories
- Preserving file permissions

## Expected API (Draft)

While the implementation is pending, the API may look similar to:

```typescript
import { createArchive, extractArchive } from '@gigadrive/build-utils';

// Create a ZIP archive
await createArchive({
  sourceDir: '/path/to/source',
  outputPath: '/path/to/output.zip',
  format: 'zip',
  compressionLevel: 9,
});

// Extract a ZIP archive
await extractArchive({
  archivePath: '/path/to/archive.zip',
  outputDir: '/path/to/output',
});
```

<Warning>
  The API shown above is a draft and may change when the feature is implemented.
</Warning>

## Current Status

As of version `1.0.1`, archive utilities are not yet exported from the package. The current exports are:

- `exec` - Command execution
- `downloadFile` - File downloads
- `getFilesForPattern` - File pattern matching

See [File Operations](/api/build-utils/file-operations) for currently available utilities.

## Workaround

Until archive utilities are implemented, you can use the underlying libraries directly:

```typescript
import archiver from 'archiver';
import extract from 'extract-zip';
import { createWriteStream } from 'fs';

// Create ZIP with archiver
const output = createWriteStream('/path/to/output.zip');
const archive = archiver('zip', { zlib: { level: 9 } });
archive.pipe(output);
archive.directory('/path/to/source', false);
await archive.finalize();

// Extract ZIP with extract-zip
await extract('/path/to/archive.zip', { dir: '/path/to/output' });
```
