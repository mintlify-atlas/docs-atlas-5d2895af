# Components

Harmony provides a comprehensive collection of production-ready React components built on Radix UI primitives with Tailwind CSS styling.

## Form Components

### Button

A flexible button component with multiple variants, sizes, and support for rendering as child elements.

```tsx
import { Button } from '@gigadrive/harmony';

<Button variant="default" size="default">
  Click me
</Button>
```

#### Props

<ParamField path="variant" type="string" default="default">
  Visual style variant: `default` | `destructive` | `outline` | `secondary` | `ghost` | `link`
</ParamField>

<ParamField path="size" type="string" default="default">
  Size variant: `default` | `sm` | `lg` | `icon`
</ParamField>

<ParamField path="wide" type="boolean" default={false}>
  When `true`, button takes full width of container
</ParamField>

<ParamField path="asChild" type="boolean" default={false}>
  When `true`, renders as Radix Slot (merges props with child element)
</ParamField>

<ParamField path="className" type="string">
  Additional CSS classes to apply
</ParamField>

#### Examples

```tsx
// Primary action button
<Button variant="default">Save Changes</Button>

// Destructive action
<Button variant="destructive">Delete Account</Button>

// Outlined button
<Button variant="outline">Cancel</Button>

// Icon button
<Button variant="ghost" size="icon">
  <IconSettings />
</Button>

// Full width button
<Button wide>Continue</Button>

// Render as link
<Button asChild>
  <a href="/dashboard">Go to Dashboard</a>
</Button>
```

### Input

Text input component with floating labels, validation, and slot support for icons.

```tsx
import { Input } from '@gigadrive/harmony';

<Input label="Email" type="email" placeholder="Enter your email" />
```

#### Props

<ParamField path="label" type="string">
  Floating label text
</ParamField>

<ParamField path="error" type="string">
  Error message (also applies error styling)
</ParamField>

<ParamField path="helpText" type="string">
  Help text displayed below input
</ParamField>

<ParamField path="leftElement" type="ReactNode">
  Icon or element displayed on the left side
</ParamField>

<ParamField path="rightElement" type="ReactNode">
  Icon or element displayed on the right side
</ParamField>

<ParamField path="bottomElement" type="ReactNode">
  Custom element displayed below input
</ParamField>

<ParamField path="type" type="string" default="text">
  HTML input type
</ParamField>

#### Examples

```tsx
// Basic input with label
<Input label="Username" />

// With error state
<Input label="Email" error="Invalid email address" />

// With help text
<Input label="Password" type="password" helpText="Must be at least 8 characters" />

// With icons
<Input 
  label="Search" 
  leftElement={<IconSearch />} 
  rightElement={<IconX />}
/>
```

## Layout Components

### Card

Container component with header, content, and footer sections.

```tsx
import { Card, CardHeader, CardTitle, CardDescription, CardContent, CardFooter } from '@gigadrive/harmony';

<Card>
  <CardHeader>
    <CardTitle>Card Title</CardTitle>
    <CardDescription>Card description goes here</CardDescription>
  </CardHeader>
  <CardContent>
    <p>Main content area</p>
  </CardContent>
  <CardFooter>
    <Button>Action</Button>
  </CardFooter>
</Card>
```

#### Components

- **Card** — Root container with border and shadow
- **CardHeader** — Header section with padding
- **CardTitle** — Title with appropriate typography
- **CardDescription** — Muted description text
- **CardContent** — Main content area
- **CardFooter** — Footer section for actions
- **CardToolbar** — Toolbar for additional controls
- **CardTable** — Wrapper for tables within cards

## Data Display

### DataTable

Full-featured table with server-side sorting, pagination, column reordering, and visibility toggling.

```tsx
import { DataTable } from '@gigadrive/harmony';

const columns = [
  { id: 'name', header: 'Name', accessorKey: 'name', sortable: true },
  { id: 'email', header: 'Email', accessorKey: 'email', sortable: true },
  { id: 'role', header: 'Role', accessorKey: 'role' },
];

<DataTable 
  columns={columns}
  data={users}
  enableColumnReordering
  enableColumnVisibility
  sorting={sorting}
  onSortingChange={setSorting}
  pagination={pagination}
  onPaginationChange={setPagination}
  totalRows={totalUsers}
/>
```

#### Props

<ParamField path="columns" type="DataTableColumn[]" required>
  Column definitions
</ParamField>

<ParamField path="data" type="TData[]" required>
  Row data
</ParamField>

<ParamField path="enableColumnReordering" type="boolean" default={false}>
  Enable drag-and-drop column reordering
</ParamField>

<ParamField path="enableColumnVisibility" type="boolean" default={false}>
  Enable column visibility toggling
</ParamField>

<ParamField path="sorting" type="SortingState | null">
  Current sorting state `{ id: string, direction: 'asc' | 'desc' }`
</ParamField>

<ParamField path="onSortingChange" type="function">
  Callback when sorting changes
</ParamField>

<ParamField path="pagination" type="PaginationState">
  Current pagination state `{ pageIndex: number, pageSize: number }`
</ParamField>

<ParamField path="onPaginationChange" type="function">
  Callback when pagination changes
</ParamField>

<ParamField path="totalRows" type="number">
  Total rows for server-side pagination
</ParamField>

<ParamField path="isLoading" type="boolean" default={false}>
  Display loading state
</ParamField>

<ParamField path="size" type="string" default="md">
  Size variant: `sm` | `md` | `lg`
</ParamField>

<ParamField path="emptyState" type="ReactNode">
  Custom empty state content
</ParamField>

<ParamField path="enablePageSizeSelector" type="boolean" default={false}>
  Show page size dropdown
</ParamField>

<ParamField path="pageSizeOptions" type="number[]" default="[10, 20, 50, 100]">
  Available page sizes
</ParamField>

### Avatar

## Component Composition Patterns

### Using `asChild`

Many components support the `asChild` prop from Radix UI's Slot component:

```tsx
// Button renders as a link
<Button asChild>
  <a href="/dashboard">Dashboard</a>
</Button>

// Works with Next.js Link
<Button asChild>
  <Link href="/profile">Profile</Link>
</Button>
```

### Combining Components

```tsx
<Card>
  <CardHeader>
    <div className="flex items-center justify-between">
      <CardTitle>Users</CardTitle>
      <Button size="sm">Add User</Button>
    </div>
  </CardHeader>
  <CardContent>
    <DataTable columns={columns} data={users} />
  </CardContent>
</Card>
```

## Styling Components

All components accept a `className` prop that's merged with default styles using the `cn()` utility:

```tsx
<Button className="mt-4 w-full">Submit</Button>
<Card className="border-2 border-primary">...</Card>
```

<Note>
Use Tailwind utility classes to customize components. The `cn()` utility uses `tailwind-merge` to intelligently merge conflicting classes.
</Note>
