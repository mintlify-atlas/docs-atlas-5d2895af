# Quick Start

This guide will get you up and running with the Gigadrive SDK in just a few minutes. We'll install the `@gigadrive/commons` package and explore some of its most useful utilities.

## Installation

<Steps>
  <Step title="Install the Package">
    First, install `@gigadrive/commons` in your project:

    <CodeGroup>
    ```bash npm
    npm install @gigadrive/commons
    ```

    ```bash yarn
    yarn add @gigadrive/commons
    ```

    ```bash pnpm
    pnpm add @gigadrive/commons
    ```

    ```bash bun
    bun add @gigadrive/commons
    ```
    </CodeGroup>
  </Step>

  <Step title="Create Your First File">
    Create a new file called `example.ts`:

    ```typescript example.ts
    import { formatFileSize, deepMerge, sha256 } from '@gigadrive/commons';

    console.log('Gigadrive SDK Quick Start');
    ```
  </Step>
</Steps>

## Common Utilities

Let's explore some of the most useful utilities in `@gigadrive/commons`.

### Format File Sizes

Convert bytes to human-readable file sizes:

```typescript
import { formatFileSize } from '@gigadrive/commons';

// Convert various file sizes
console.log(formatFileSize(1234));          // "1.21 KB"
console.log(formatFileSize(123456789));     // "117.74 MB"
console.log(formatFileSize(1073741824));    // "1 GB"
console.log(formatFileSize(0));             // "0 B"
```

<Tip>
  `formatFileSize` is perfect for displaying upload progress, file listings, or storage quotas in your UI.
</Tip>

### Deep Merge Objects

Merge multiple objects deeply, with array concatenation:

```typescript
import { deepMerge } from '@gigadrive/commons';

const defaults = {
  theme: 'light',
  features: ['auth', 'api'],
  config: {
    timeout: 3000,
    retries: 3
  }
};

const userSettings = {
  theme: 'dark',
  features: ['analytics'],
  config: {
    timeout: 5000
  }
};

const merged = deepMerge(defaults, userSettings);

console.log(merged);
// Output:
// {
//   theme: 'dark',                              // overwritten
//   features: ['auth', 'api', 'analytics'],    // concatenated
//   config: {
//     timeout: 5000,                           // overwritten
//     retries: 3                               // preserved
//   }
// }
```

<Note>
  `deepMerge` supports multiple partial objects and safely handles prototype pollution by ignoring `__proto__` and `constructor` keys.
</Note>

### Hash Strings with SHA-256

Generate SHA-256 hashes for strings:

```typescript
import { sha256 } from '@gigadrive/commons';

const hash = await sha256('Hello, Gigadrive!');
console.log(hash);
// Output: "a3c4d5e6f7... (64-character hex string)"
```

### Encrypt and Decrypt Data

Securely encrypt and decrypt strings using AES-GCM:

```typescript
import { encrypt, decrypt } from '@gigadrive/commons';

const secretKey = 'my-secret-key';
const message = 'Sensitive data';

// Encrypt
const encrypted = await encrypt(message, secretKey);
console.log('Encrypted:', encrypted);
// Output: "SGVsbG8gV29ybGQh..." (base64 string)

// Decrypt
const decrypted = await decrypt(encrypted, secretKey);
console.log('Decrypted:', decrypted);
// Output: "Sensitive data"
```

<Tip>
  The encryption uses AES-GCM with a 256-bit key derived from your password via SHA-256. Each encryption generates a random IV for maximum security.
</Tip>

### Sleep Utility

Add delays with a Promise-based sleep function:

```typescript
import { sleep } from '@gigadrive/commons';

async function example() {
  console.log('Starting...');
  await sleep(2000); // Wait 2 seconds
  console.log('2 seconds later!');
}
```

## Complete Example

Here's a complete example that demonstrates multiple utilities:

<CodeGroup>
```typescript example.ts
import {
  formatFileSize,
  deepMerge,
  sha256,
  encrypt,
  decrypt,
  sleep
} from '@gigadrive/commons';

async function demo() {
  console.log('=== Gigadrive SDK Demo ===\n');

  // 1. Format file sizes
  console.log('File Sizes:');
  console.log(`  ${formatFileSize(1234567)}`);
  console.log(`  ${formatFileSize(9876543210)}\n`);

  // 2. Deep merge configuration
  const config = deepMerge(
    { api: { timeout: 3000 }, features: ['auth'] },
    { api: { retries: 5 }, features: ['logging'] }
  );
  console.log('Merged Config:', config);
  console.log();

  // 3. Hash a string
  const hash = await sha256('gigadrive-sdk');
  console.log('SHA-256 Hash:', hash.substring(0, 16) + '...');
  console.log();

  // 4. Encrypt and decrypt
  const secret = 'my-encryption-key';
  const data = 'Hello, Gigadrive!';
  const encrypted = await encrypt(data, secret);
  const decrypted = await decrypt(encrypted, secret);
  
  console.log('Encryption:');
  console.log(`  Original: ${data}`);
  console.log(`  Encrypted: ${encrypted.substring(0, 20)}...`);
  console.log(`  Decrypted: ${decrypted}`);
  console.log();

  // 5. Sleep
  console.log('Waiting 1 second...');
  await sleep(1000);
  console.log('Done!');
}

demo().catch(console.error);
```

## Common Patterns

### Configuration Management

```typescript
import { deepMerge } from '@gigadrive/commons';

function loadConfig(userConfig = {}) {
  const defaults = {
    environment: 'production',
    port: 3000,
    features: {
      auth: true,
      logging: true
    }
  };

  return deepMerge(defaults, userConfig);
}
```

### Data Validation

```typescript
import { sha256 } from '@gigadrive/commons';

async function verifyIntegrity(data: string, expectedHash: string) {
  const hash = await sha256(data);
  return hash === expectedHash;
}
```

### Secure Storage

```typescript
import { encrypt, decrypt } from '@gigadrive/commons';

class SecureStorage {
  constructor(private key: string) {}

  async save(data: string): Promise<string> {
    return await encrypt(data, this.key);
  }

  async load(encrypted: string): Promise<string> {
    return await decrypt(encrypted, this.key);
  }
}
```

<Note>
  All utilities in `@gigadrive/commons` are tree-shakeable, so your bundler will only include the functions you actually use.
</Note>
